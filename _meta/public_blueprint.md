# Psychology Money Reader v11 - Public Blueprint

## Overview
A mobile-first, offline-capable bilingual reader for *The Psychology of Money* and *Positive Psychology Progress* research paper.

## Repository & Live URL
- Public GitHub Repository: https://github.com/Yunkumom/reader
- GitHub Pages Live Site: https://yunkumom.github.io/reader/

## Key Features
- Progressive Section-by-Section Loading for mobile speed.
- IndexedDB Local Device Storage (`ReaderOfflineCacheDB`) for zero-bandwidth offline reading.
- Full bilingual English/Chinese alignment with TTS speech synthesis.
- Figures and tables rendering for research papers.
