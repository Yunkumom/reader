🗺️ System Map｜系統架構圖與模組關係

1. 專案根目錄入口 (Root Executable Entry)

index.html - 單檔獨立雙語閱讀器，支援 IndexedDB 離線快取、漸進式載入、PWA 主畫面安裝、全篇搜尋、書籤與 TTS 語音朗讀。

2. 資料夾分類架構 (Organized Directory Layout)

  • source_materials/ - 原始書籍與學術論文 JSON / 圖表原始檔案。
  • processed_content/ - 中英文對照段落內容，採純 Markdown (.md) 格式儲存。
  • reader_versions/ - 閱讀器版本演進紀錄與可用建置 (包含 v11_latest, v10_pwa_offline, v9_bilingual_figures)。
  • _meta/ - 擁有者私有與公開可重建藍圖 (owner_private_blueprint.md, public_blueprint.md)。
  • _agent/ - AI Agent 工作日誌與指令規範。
  • _human/ - 人工維護層 (系統圖譜與常見問題快速排除)。
  • _pending/ - 封存資料與待審查暫存區。
